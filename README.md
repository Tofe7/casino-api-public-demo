# 🎰 Casino Games Aggregator – Demo / Showcase

> **No real money, no sports betting – just a demo/ showcase of aggregated casino games (slots, live casino, roulette, etc.) from providers such as Pragmatic Play, Evolution, and others.**

This project is a **public demo** that displays a large collection of casino‑style games. All game data (thumbnails, names, demo links) is provided by a Public API server. The frontend consumes this privet ID to present the games in a modern dashboard.

**No provider contact, just public API keys and private ID, and no real MONEY** – everything is for demonstration purposes only.

---

## 🧩 Features

- **Slots, live casino, roulette, table games** – aggregated from multiple providers (Pragmatic Play, Evolution, and others).
- **Responsive dashboard** – browse games by category.
- **Demo mode** – each game launches a free‑to‑play demo (user login, admin deposit).

## 🚀 Getting started

### 1. Clone or download the repository

1 - git clone https://github.com/Tofe7/casino-api-public-demo.git
2 - cd casino-api-public-demo
3 - python api.server.py
The API will run on http://localhost:5000 (or the port defined in the script).
4 - npm install


Option A – Manual start (separate terminals)
🐍 Start the API server
Open a terminal in the project root and run:

python api.server.py
The API will run on http://localhost:5000 (or the port defined in the script).

🌐 Start the frontend
Open another terminal in the same folder and run:

npm start
The frontend will open at http://localhost:3000 (or the port shown in the console).

Option B – Auto‑run with the batch file (Windows only)
Double‑click Auto-Run
