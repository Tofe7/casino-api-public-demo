# 🎰 Casino Games Aggregator – Demo / Showcase

> **No real money, no sports betting – just a demo/ showcase of aggregated casino games (slots, live casino, roulette, etc.) from providers such as Pragmatic Play, Evolution, and others.**

This project is a **public demo** that displays a large collection of casino‑style games. All game data (thumbnails, names, demo links) is provided by a local API server. The frontend consumes this API to present the games in a modern dashboard.

**No provider contact, just public API keys and private id, and no real gambling DEMO** – everything is for demonstration purposes only.

---

## 🧩 Features

- **Slots, live casino, roulette, table games** – aggregated from multiple providers (Pragmatic Play, Evolution, and others).
- **Responsive dashboard** – browse games by category.
- **Demo mode** – each game launches a free‑to‑play demo (user login, admin deposit).
- **One‑click start** – use the provided batch file to setup the API and the frontend together.
